# IPL Dashboard 2025

## 📊 Project Overview
This project is an interactive Excel dashboard built using IPL dataset.

It provides insights into:
- Total Runs
- Total Wickets
- Team Performance
- Venue Analysis

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## 📌 Features
- Dynamic filtering using slicers
- Interactive charts
- Clean dashboard layout
